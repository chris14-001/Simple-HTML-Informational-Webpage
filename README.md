# Informational HTML & CSS Webpage

## Overview
This project is a simple informational webpage built using **HTML and CSS**. It is designed to teach and demonstrate the fundamentals of HTML structure, semantic elements, forms, media handling, accessibility, and basic CSS styling.

The webpage serves as a beginner-friendly guide to understanding how modern websites are structured and styled.

---

## Features

- Clean and responsive layout
- Navigation bar with anchor links
- Sections covering:
  - HTML document structure
  - Semantic HTML
  - Attributes and links
  - Images and media usage
  - Forms and inputs
  - Accessibility fundamentals
- Embedded YouTube video for learning support
- Interactive feedback form (example)
- Social media/contact icons (GitHub, Instagram, Email)
- Styled with modern CSS (Flexbox, hover effects, gradients)

---

## Technologies Used

- HTML5
- CSS3

## Key Learning Goals

This project helps you understand:

- How HTML documents are structured
- The importance of semantic elements
- How forms collect user input
- How accessibility improves user experience
- How CSS is used to style and layout content
- How Flexbox is used for alignment and spacing
